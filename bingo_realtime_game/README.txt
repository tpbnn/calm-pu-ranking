실시간 빙고 게임 v3

파일 구성:
- bingo.html : 방장/참가자 통합 페이지
- firebase-config.js : Firebase 설정 파일

v3 수정사항:
1. 빙고판 타이틀/요소 구성 변경
   - 1줄: 빙고판 이름
   - 2줄: 빙고 개수 / 보유 치킨 / 리셋
   - 3줄: 치킨 +1 / 치킨사용 ON/OFF / 지우개 ON/OFF
2. 입력 기록 접기/펼치기 기능 추가
   - 기본값: 접힌 상태
   - 방장/참가자 모두 적용
3. 숫자 복수 입력 지원
   - 예: 3, 17, 45
   - 이미 처리된 숫자, 빙고판에 없는 숫자, 중복 숫자는 자동으로 건너뜀
4. 지우개 기능 추가
   - 지우개 ON 상태에서 이미 입력/치킨 처리된 숫자를 클릭하면 해제됨

사용 방법:
1. firebase-config.js에 기존 Firebase 설정값 입력
2. bingo.html을 Live Server 또는 GitHub Pages에서 실행
3. 방장 모드에서 방 ID 입력 후 빙고판 생성
4. 참가자는 같은 페이지에서 참가자 모드 선택 후 같은 방 ID 입력
